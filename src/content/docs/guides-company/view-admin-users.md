## Manage Admin Panel Users (View Only)

Esta sección te permite visualizar todos los usuarios administradores que tienen acceso al panel de administración de MagnetoHR. Desde aquí puedes consultar quiénes son los administradores, sus roles y quién los agregó al sistema.

### Paso 1: Acceder a la Sección de Usuarios Administradores

Colocáte en la sección **Manage admin panel users (View Only)** (Gestionar usuarios del panel de administración - Solo Vista):

![Sección de gestión de usuarios administradores](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/4ce43be69f466959216c2d905effa937/screenshot/54d517c7-a7d2-4ee4-9a3c-b24e5d41b9ff.png)

### Paso 2: Revisar la Tabla de Usuarios

Dentro de esta sección podrás observar una tabla con la siguiente información sobre cada usuario administrador:

![Tabla de usuarios administradores](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/36587fe59c0ebaf500ddc1de5bed478b/screenshot/2d9733d2-96a9-437f-a5a2-3fdbd19eb71d.png)

**Campos de la tabla:**

- **Usuario (User)**: Información del usuario administrador (nombre y correo electrónico)
- **Rol (Role)**: Rol o permisos asignados al usuario dentro del panel
- **Agregado Por (Added By)**: Usuario administrador que agregó a este usuario al sistema

### Nota Importante

**Nota:** Los nuevos usuarios se agregan directamente a la base de datos. Los usuarios reciben un correo electrónico de bienvenida con credenciales temporales para acceder al panel de administración.
